Hello! Welcome to my API Project for Career Foundry. This api will hold favorite movies as instructed in the course. However I will app on a search feature and styling later on down the line as a test of my skills.

  • Return a list of all movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., "Thriller")
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username)
  • Allow users to add a movie to their list of favorites (showing only a text that a movie has been added-more on this later)
  • Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed-more on this later)
  • Allow existing users to deregister (showing only a text that a user email has been removed-more on this later)

API Endpoint Documentaion

Buisness Logic Endpoint Method Query Parameters Format of data expected in request and an example Format of data expected in response
Returns list of all movies /movies GET JSON object holding list of all movies
Returns movie info by title /movies/:title GET :title JSON object containing all info about chosen movie
Returns info about genre by movie title /movie/genres/:title GET :title JSON object containing the genre of the chosen movie
Returns director info /movies/directors GET :name JSON object containing all info about the director
Registers new user /users POST JSON object about the user to be created, example: {"name": "Seth"} JSON object containing info about the user created
Updates a users info /users/:id PUT :id JSON object containing updated user info, example:{"name":"Jason"} JSON object containing the new user info
Allows users to add a movie to their favourites /users/:id/:movieTitle POST :id, :movieTitle - A JSON object holding data about the user and the movie that was added
Allows users to remove a movie from their favourites /users/:id/:movieTitle DELETE :id, :movieTitle - A JSON object holding data about the user and the movie that was removed
Allows existing users to deregister /users/:id DELETE :id - A text message indicating whether the user has successfully deregistered
